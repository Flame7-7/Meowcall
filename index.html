About MeowCall

The Bot

MeowCall is a Discord bot designed to help servers connect with each other through voice and text channels. It was created to help communities grow and make new friends in a safe, moderated environment.

With features like random server connections, fun games, GIF support, and safety tools, MeowCall provides a unique way to expand your Discord experience.

Features:

  • Random server connections
  • GIF support (Tenor/Giphy)
  • Multiple fun games
  • User statistics and leaderboards
  • Webhook-based messaging
  • Safety and moderation tools

Legal Information

Privacy Policy

Last Updated: April 7, 2025

MeowCall collects the following information:

  • Discord user IDs of users who interact with the bot
  • Server IDs where the bot is installed
  • Channel IDs where bot commands are used
  • Command usage statistics

We use this information to provide and improve our service, monitor for abuse, and generate anonymous usage statistics.

All data is stored securely. User IDs and server IDs are retained indefinitely for moderation purposes. Message content is only stored temporarily during active connections.

Terms of Service

By using MeowCall, you agree to:

  • Not harass other users
  • Not share inappropriate content
  • Not attempt to disrupt the service
  • Not use the bot for illegal activities

We reserve the right to ban users/servers that violate these terms, monitor connections for quality control, and modify or terminate the service at any time.

MeowCall is not responsible for content shared through the bot by users or disputes between connected servers.

Data Deletion Request

To request deletion of your data, please join our support server and contact the bot developers. We will process your request within 30 days.

Note that some data may be retained for legal or security reasons even after deletion requests.
