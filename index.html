Commands

Connection Commands

/call
Connect your server to another random server
/skip
Skip the current connection and find a new server
/hang
End the current connection

Social Commands

gif
Share animated GIFs in connected chats (supports Tenor/Giphy links!)
/friendrequest
Send your Discord username to the connected server
/flip
Flip a coin with fun animations
/ship
Calculate compatibility between two users

Game Commands

/smashorpass
Play Smash or Pass with random anime characters
/8ball
Ask the magic 8-ball a yes/no question
/random
Randomly ship two server members
/truthdare
Play Truth or Dare with your friends

Info Commands

/mystats
Check your call statistics
/help
Show all available commands
/ping
Check bot latency
/leaderboard
View top users/servers with most connections

Privacy Policy

Last Updated: April 7, 2025

1. Information We Collect

MeowCall collects the following information when you use our bot:

  • Discord user IDs of users who interact with the bot
  • Server IDs where the bot is installed
  • Channel IDs where bot commands are used
  • Message content when using the connection features
  • Command usage statistics

2. How We Use Your Information

We use the collected information to:

  • Provide and maintain our service
  • Improve user experience
  • Monitor for abuse and enforce our Terms of Service
  • Generate anonymous usage statistics

3. Data Storage

All data is stored securely on our servers. We retain:

  • User IDs and server IDs indefinitely for moderation purposes
  • Message content only temporarily during active connections
  • Command usage data for 30 days
  • We publicly show the servers Meowcall is in for leaderboards command

4. Your Rights

You have the right to:

  • Request access to your personal data
  • Request correction or deletion of your data
  • Opt-out of data collection (by not using the bot)

5. Contact Us

For privacy-related concerns, please join our support server and contact the bot developers.
